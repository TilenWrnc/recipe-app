Recipe App

A simple application built with Next.js that allows users to explore meal categories, browse recipes, and view detailed cooking instructions. The app fetches data from TheMealDB API, providing a rich collection of recipes from various cuisines.

Made using: 
  - Next.js 
  - React 
  - Tailwind CSS 
  - TypeScript 
  - TheMealDB API - Open API for meal and recipe data
